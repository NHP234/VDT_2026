# Workspace Slides - Slide Thuyết Trình

Dùng thư mục này cho presentation khoảng 15 slide.

Flow khuyến nghị:

1. Vấn đề.
2. Mục tiêu dự án.
3. Phạm vi MVP.
4. Kịch bản demo cuối.
5. Ranh giới hệ thống.
6. Tổng quan kiến trúc.
7. Unified domain model.
8. Luồng inbound Facebook.
9. Luồng inbound email.
10. Luồng reply và delivery result.
11. Reliability: idempotency, retries và DLQ.
12. Security và privacy.
13. Bằng chứng test.
14. Screenshots demo.
15. Giới hạn và bước tiếp theo.

Giữ screenshots không nhạy cảm và nhất quán với deterministic demo data.
