# Truy Vết Requirement

Mọi implementation task, API, table, event, UI control và automated test phải
tham chiếu ít nhất một locked requirement từ `docs/requirement.md`.

Giá trị trạng thái:

- `Planned`
- `Implemented`
- `Verified`
- `Deferred`
- `Blocked`
